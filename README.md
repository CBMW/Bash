# Bash Scripting
Bash scripting tools and automation

  -Cleans system with apt autoremove and apt clean after updating

## WSCap.sh  
  -Easily save capture files from terminal  
  -Set wifi interface
  -Set duration of the capture  
  -Set location to save capture file
